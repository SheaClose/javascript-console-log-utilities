# convert-input-to-get-set

Select an @Input() decorator in angular and convert it to getter/setter functions

# Installing

This extension is available for free in the [Visual Studio Code Marketplace](https://marketplace.visualstudio.com/items?itemName=sheaclose.convert-input-to-get-set)

# Usage

Select @Input property and type, or just the property name and execute :
Shift + Win + J
Shift + Cmd + J
